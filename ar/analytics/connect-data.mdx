## إعدادات الاتصال

انتقل إلى **Analytics → Data sources → Add source** وأكمل الحقول التالية:

| الحقل | الوصف |
|-------|-------------|
| **Source name** | تسمية يمكن للبشر قراءتها وتستخدم في واجهة المستخدم. |
| **Source type** | اختر من القائمة المدعومة أعلاه. |
| **Host / URL** | اسم المضيف أو عنوان URL لنقطة النهاية الخاصة بالمصدر. |
| **Port** | المنافذ الافتراضية معبأة مسبقًا للمصادر الشائعة. |
| **Database / Schema** | قاعدة البيانات أو المخطط المحدد للاتصال به. |
| **Username / API key** | بيانات اعتماد المصادقة للمصدر. |
| **Password / Secret** | يتم تخزينها مشفرة ولا تُعرض أبدًا بعد الحفظ. |
| **SSL mode** | اشتراط أو تفضيل أو تعطيل SSL. اشترط SSL لمصادر الإنتاج. |

انقر على **Test connection** للتحقق من بيانات الاعتماد قبل الحفظ. تتضمن الأخطاء رمز سبب محددًا للمساعدة في استكشاف مشكلات جدار الحماية أو بيانات الاعتماد وإصلاحها.

## وتيرة المزامنة

بعد توصيل مصدر، قم بتكوين عدد المرات التي يتم فيها تحديث البيانات:

- **Real-time** — يتم تشغيل الاستعلامات بشكل مباشر على المصدر في كل مرة يتم فيها تحميل لوحة المعلومات. الأفضل للمصادر منخفضة الكمون مع استعلامات مفهرسة.
- **Scheduled sync** — يتم جلب البيانات على فترة زمنية ثابتة (15 دقيقة، أو ساعة، أو 6 ساعات، أو يوميًا) وتخزينها مؤقتًا. الأفضل لمجموعات البيانات الكبيرة أو واجهات API ذات معدلات محدودة.
- **Manual** — التحديث فقط عند النقر على **Sync now**. مناسب لعمليات استيراد البيانات لمرة واحدة.

<Info>
  تعتمد فترات المزامنة المجدولة على خطتك. تدعم الخطط المجانية المزامنة اليومية فقط.
</Info>
